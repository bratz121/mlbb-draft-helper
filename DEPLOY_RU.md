# Деплой без VPN для России

Если сайт открывается только с VPN, а картинки уже лежат локально в `assets/heroes`, проблема чаще всего не в коде, а в доступности домена хостинга.

## Что проверить

1. Убедись, что на Netlify загружена вся папка проекта, включая:
   - `index.html`
   - `styles.css`
   - `app.js`
   - `assets/heroes`
2. Открой сайт и проверь версию в правом верхнем углу.
3. Если страница не грузится без VPN вообще, вероятно блокируется/плохо резолвится домен `netlify.app`.

## Более надежные варианты

Для аудитории из России лучше протестировать несколько зеркал:

- GitHub Pages
- Cloudflare Pages
- Timeweb Cloud Static
- Selectel Object Storage + CDN
- VK Cloud Static Website
- обычный VPS с Nginx

## Самый надежный подход

Купить домен и направить его на хостинг, который стабильно открывается у твоей аудитории.

Проект статический, поэтому ему не нужен серверный код. Достаточно отдавать файлы как обычный сайт.
