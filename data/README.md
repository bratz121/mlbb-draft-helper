# Data layer plan

Следующий шаг для чистой архитектуры: вынести данные из `app.js` в JSON-файлы.

Планируемая структура:

- `heroes.json` — герои, роли, tier, meta, заметки
- `counters.json` — взвешенные контрпики
- `synergies.json` — синергии героев
- `stage-bias.json` — сила героев по стадиям игры
- `portraits.json` — локальные пути к портретам

Пока данные остаются в `app.js`, чтобы не усложнять деплой и не добавлять асинхронную загрузку JSON на статическом сайте.
